# Usage Pattern

## Get Started
Build an index from documents:
```python
from llama_index import GPTVectorStoreIndex

index = GPTVectorStoreIndex.from_documents(docs)
```

> Note: To learn how to load documents, see [Data Connectors](/how_to/connector/root.md)
